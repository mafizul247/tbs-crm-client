# 🧮 Tax Calculator Bangladesh (2026–2027)

A modern, responsive web application to calculate income tax in Bangladesh based on the latest tax rules. Built with React, Tailwind CSS, and DaisyUI, with multi-language support (English & বাংলা).

## 📊 How It Works

1. Select your **Tax Category**
   - General Person  
   - Female / Senior Citizen  
   - Disabled Person  
   - Freedom Fighter  

2. Select your **City Corporation**
   - Dhaka / Chattagram  
   - Other City  
   - Rural Area  

3. Enter your **Annual Gross Salary**

4. Click **Calculate Tax**

👉 The app will show:
- Tax exemption
- Taxable income
- Total tax
- Monthly tax
- Minimum tax (based on location)
